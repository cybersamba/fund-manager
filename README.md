# Fund Manager

Aplicación de gestión de fondos de inversión construida con React, Vite y TailwindCSS.

## Requisitos Previos

Necesitas tener **Node.js** instalado en tu sistema para ejecutar esta aplicación.
Puedes descargarlo desde: [https://nodejs.org/](https://nodejs.org/)

## Instalación y Ejecución

Una vez que tengas Node.js instalado, abre una terminal en esta carpeta y ejecuta los siguientes comandos:

1.  **Instalar dependencias:**
    ```bash
    npm install
    ```

2.  **Iniciar el servidor de desarrollo:**
    ```bash
    npm run dev
    ```

3.  Abre tu navegador en la URL que aparecerá en la terminal (generalmente `http://localhost:5173`).

## Características

- Registro de compras y ventas.
- Cálculo automático del valor del portafolio.
- Persistencia de datos en el navegador (Local Storage).
- Diseño moderno y responsivo.
